Meu Portfólio - Marlon Schulze
Este é o repositório do meu portfólio pessoal, onde apresento meus projetos, habilidades e informações de contato. O site é responsivo e otimizado para dispositivos móveis, com navegação intuitiva e design moderno.

---

Tecnologias Utilizadas
HTML5

CSS3 (Flexbox, Grid, animações e responsividade)

JavaScript (interatividade no menu hamburger e outros detalhes)

AOS (Animate On Scroll) para animações ao rolar a página

---

Funcionalidades
Menu hamburger responsivo para dispositivos móveis

Animações suaves no header e links

Seções de apresentação, habilidades, projetos e contato

Design adaptado para mobile, tablet e desktop
